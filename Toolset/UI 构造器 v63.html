物理弹簧

⚙️ 弹簧物理参数配置

📘 物理模式说明
• 弹簧强度 (K): 回弹力 = 目标位置差 × K。值越大回弹越快。
• 阻尼系数 (D): 速度衰减 = 当前速度 × D。值越接近 1，振荡越持久。
• 普通弹簧: 直接围绕默认目标或自动目标做回弹。
• 胸部抖动模式: 仅用于摇杆。拖拽先推动隐藏驱动点，真实输出延迟跟随，所以会有更软的滞后、回弹和释放抖动。
• 边界反弹: 控制撞到边界后的回弹保留量。越小越稳，越大越活跃。
📘 自动动画 / 函数说明
• 自动动画: 支持物理的组件会生成独立的 $auto_x 开关，面板勾选只决定默认值。
• 轨迹来源: 可选混沌轨迹或自定义函数。
• 轨迹刷新 / 函数周期: 混沌模式表示多少帧刷新一次目标，函数模式表示完整跑完整个函数要多少帧。
• 动画速度 / 目标平滑: 限制内部目标每帧最多移动多少，并控制靠近目标时的柔和度。
• 轨迹种子: 控制自动轨迹起始相位。X/Y 使用不同种子可以避免长期锁成一条对角线。
• 整体优先级: 默认目标 / 自动目标 / 拖拽输入 三层叠加，拖拽优先，自动动画只是附加目标。
算法说明:
• 弹簧强度 (K): 回弹力 = 目标位置差值 × K。值越大回弹越快。
• 阻尼系数 (D): 速度衰减 = 当前速度 × D。值越接近1，振荡越持久。
• 自动动画: 支持物理的组件会生成独立的 $auto_x 开关，面板里的勾选只决定它的默认值。运行时把 $auto_x 设为 1 即开启，设为 0 即关闭。
• 轨迹来源: 可选混沌轨迹或自定义函数。自定义函数支持 t，范围为 0~1。
• 轨迹刷新 / 函数周期: 在混沌模式下表示隔多少帧刷新一次目标；在函数模式下表示完整跑完一圈函数需要多少帧。数值越大越慢。
• 动画速度 / 目标平滑: 动画速度限制内部目标每帧最多移动多少，目标平滑决定它向自动目标靠近时有多柔和。两者一起用来避免瞬移和闪动。
• 轨迹种子: 用来控制自动轨迹的起始相位。X/Y 使用不同种子和不同混沌参数，避免两个轴长期锁成同一条对角线。
• 胸部抖动模式: 仅用于摇杆。拖拽时先推动一个隐藏目标点，真实输出通过弹簧延迟跟随，所以会有更柔软的滞后、回弹和释放抖动。
• 边界反弹: 控制撞到边界后的回弹保留量。越小越稳，越大越活跃。
• 公式: 默认目标 / 自动目标 / 拖拽输入 三者分层处理。拖拽优先，自动动画只是附加目标，不会抢走组件的正常交互。
胸部抖动模式仅用于摇杆。拖拽时会先驱动目标点，真实输出会带滞后、回弹和释放后的抖动，更适合做软组织感的上下晃动。
🎞️ 自动动画参数
🧮 自定义函数语法
支持变量 t，范围为 0~1。
输出范围说明: 摇杆函数建议返回 -1 ~ 1，普通滑块函数建议返回 0 ~ 1。
摇杆整圆示例: X(t)=sin(TAU*t), Y(t)=cos(TAU*t)。
想让轨迹更大: 直接把函数乘大，例如 1.6*sin(TAU*t) / 1.6*cos(TAU*t)，或者提高“横向幅度 / 纵向幅度”。
支持运算符: + - * / % < <= > >= == != && || ! ?:
可用常量: PI TAU E
可用函数: sin cos tan asin acos atan atan2 abs min max pow sqrt cbrt log log2 exp floor ceil round sign clamp fract mix lerp step smoothstep saw tri pingpong sin01 cos01 mod repeat saturate invlerp pulse select bezier3 bezier4。
这些函数只在生成器内部解析和采样，导出的 3DM INI 不会原样写出任何函数调用。
函数曲线预览
0° 位于正上方，顺时针递增。方向中心角 = 起始角度 + n × (360 / 方向数量)。一级细分时会按摇杆当前位置对相邻方向做连续融合，中间为 0，沿对应方向推到边界时为 1；多级细分会在每个方向的连续权重上继续做累计细分。
